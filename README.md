# Task Manager API

API RESTful desenvolvida para o gerenciamento de tarefas, permitindo criar, listar, atualizar e remover registros por meio de operações CRUD. O projeto foi criado com foco em aprendizado de APIs REST, organização em camadas e boas práticas no desenvolvimento back-end.

## Funcionalidades
- Criar tarefas
- Listar tarefas
- Atualizar tarefas
- Remover tarefas
- Respostas HTTP padronizadas

## Tecnologias Utilizadas
- Java
- Spring Boot
- API REST
- Maven
- H2
- MySQL
